# CS2 Farmer Panel

Панель для автоматического фарма CS2 с поддержкой maFiles, Anti-AFK, режимами счёта и статистикой дропа.

## Возможности
- Импорт аккаунтов из файла (логин:пароль)
- Автоматический вход через maFiles
- Проверка банов через Steam API
- Запуск нескольких аккаунтов одновременно
- Создание двух лобби в режиме Wingman (2x2) и поиск матча друг против друга
- Anti-AFK для предотвращения кика за бездействие
- Три режима счёта: 8:8 (ничья), random, simulation
- Сбор дропа и подробная статистика (сегодня, неделя, месяц, всего) с интерактивными графиками
- Шифрование всех паролей мастер-паролем
- Современный интерфейс с иконками, подсказками и цветовой индикацией

## Требования
- Windows 10/11
- Python 3.11 (для разработки) или собранный .exe
- Node.js (если запуск из исходников)

## Установка (из исходников)
1. Установите Python 3.11 и Node.js.
2. В папке `src/node` выполните `npm install`.
3. В корневой папке выполните `pip install -r requirements.txt`.
4. Запустите `python -m src.python.main`.

## Сборка .exe
Выполните `python build.py`. Готовый файл будет в папке `dist`.

## Безопасность
- Мастер-пароль не хранится в открытом виде, используется для шифрования.
- Логи шифруются.
- Передача данных между Python и Node.js через временные файлы.
